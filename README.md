**GuildGate** è una **web-app** progettata per connettere i giocatori nel nostro universo MMORPG. Questa piattaforma permette agli utenti di **creare**, **gestire** e **unirsi a party** in modo rapido e organizzato, con requisiti di classe, livello, server e obiettivi di gioco (Boss, Dungeon).
